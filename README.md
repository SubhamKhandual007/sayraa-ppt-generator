# 🎯 SuuSri AI PPT Generator

An AI-powered Presentation Generator built with Python and Flask that automatically creates professional PowerPoint presentations from user topics and prompts.

🌐 Live Demo: https://ppt-generator-seven-mu.vercel.app

---

## 🚀 Features

- 🤖 AI-powered slide generation
- 📄 Create presentations from custom topics
- 🎨 Beautiful and responsive UI
- ⚡ Fast PPT generation
- 📥 Download PowerPoint presentations
- ☁️ Vercel deployment support
- 🔐 Secure environment variable handling
- 📱 Mobile-friendly interface
